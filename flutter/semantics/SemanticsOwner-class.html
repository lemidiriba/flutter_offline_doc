SemanticsOwner
description

SemanticsOwner class Null safety

Owns SemanticsNode objects and notifies listeners of changes to the render tree semantics.

To listen for semantic updates, call PipelineOwner.ensureSemantics to obtain a SemanticsHandle. This will create a SemanticsOwner if necessary.
